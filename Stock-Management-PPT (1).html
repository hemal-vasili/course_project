Technical Foundation

Data Structure

typedef struct {
  int id;
  char name[100];
  char company[100];
  int price;
  int quantity;
} Product;

Each product is represented with 5 key attributes for complete inventory tracking.

Data Persistence

  • Binary File I/O - Fast and compact storage
  • fopen/fwrite/fread - C standard library functions
  • stock.dat - Persistent data file
  • Data survives program exits and re-runs
  • Efficient storage format for large inventories

Program Execution Flow

  1. Load existing products from file
  2. Display menu with 6 options
  3. Process user choice:
    • Add ‚Üí Validate ID ‚Üí Save to file
    • View ‚Üí Load and display all records
    • Search ‚Üí Find by ID ‚Üí Display details
    • Update ‚Üí Find ‚Üí Modify ‚Üí Rewrite file
    • Delete ‚Üí Find ‚Üí Remove ‚Üí Rewrite file
  4. Loop until user exits
